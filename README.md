# Vikk Todo
Todo list app made with Vikk

## Demo :gem:
[localboards.ggez.se](https://localboards.ggez.se)

## Features :clipboard:
- Stores all data in localstorage
- Download/Upload option
- Light/Dark mode

## Stack :zap:
- HTML, CSS, TS
- [Vikk](https://github.com/vikkjs/vikk)
- Vite
